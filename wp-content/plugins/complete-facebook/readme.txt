== Description ==

With this plugin you can Enable Like/Share/Send button to a specific post/page or for all your posts/page or both! You can enable notifications via e-mail. Show the buttons on the index page and more. 

Features:

* Like Button
* Comment Button
* Like Notifcation
* Comment Notification
* Widget
* OpenGraph Metadata 


== Installation ==

1. Download the zip file
2. Upload it to your wordpress site
3. Activate the plugin
4. Check the options, you can find options under 'Settings' & 'Plugins' from left sidebar
5. Check the Widget.
